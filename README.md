# Novan Portfolio

Portfolio website built with React + Vite.

## Fitur

- **7 Halaman**: Home, About, Projects, Blog, CV, Contact, 404
- **Dark Mode**: Toggle dark/light theme
- **Animasi**: Page transitions, typing effect, parallax scroll, stagger cards
- **Custom Cursor**: Shooting star trail effect
- **Background**: Particle network + grid pattern
- **Responsif**: Mobile-friendly dengan hamburger menu
- **Scroll Progress Bar**: Indikator progres scroll
- **Form Kontak**: Fungsional dengan Formspree
- **GitHub API**: Projects otomatis dari repo asli
- **SEO**: Meta tags + Open Graph tiap halaman
- **Animated Navbar**: Spring animation indicator

## Tech Stack

- React 19 + Vite
- React Router DOM
- Framer Motion (animasi)
- Canvas API (partikel + cursor)

## Cara Menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:5173

## Build

```bash
npm run build
```

Hasil build di folder `dist/`.

## Deploy

**Vercel** (otomatis):
```bash
npx vercel
```

**Netlify** (drag & drop):
- Upload folder `dist/` ke Netlify

Atau hubungkan repo GitHub langsung ke Vercel/Netlify untuk auto-deploy.
