# Novan Portfolio

Portfolio website built with React + Vite.

---

## Features

- **7 Pages**: Home, About, Projects, Blog, CV, Contact, 404
- **Dark Mode**: Toggle dark/light theme with localStorage persistence
- **Animations**: Page transitions, typing effect, parallax scroll, stagger cards
- **Custom Cursor**: Shooting star trail effect that follows your mouse
- **Background**: Particle network + subtle grid pattern
- **Responsive**: Mobile-friendly with hamburger menu
- **Scroll Progress Bar**: Gradient bar at the top of the page
- **Contact Form**: Functional with Formspree integration
- **GitHub API**: Projects auto-fetched from real GitHub repositories
- **SEO**: Meta tags + Open Graph for every page
- **Animated Navbar**: Spring-based animated underline indicator

## Tech Stack

- React 19 + Vite
- TypeScript
- React Router DOM
- Framer Motion (animations)
- Canvas API (particles + cursor)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

Output in `dist/` folder.

## Deploy

**Vercel** (automatic):
```bash
npx vercel
```

**Netlify** (drag & drop):
- Upload `dist/` folder to Netlify

Or connect your GitHub repo directly to Vercel/Netlify for auto-deploy.
